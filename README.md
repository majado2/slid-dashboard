# SLID Dashboard (Vite + React + TypeScript)

لوحة تحكم حكومية لإدارة طلبات التتبع والمستفيدين، مع خريطة تفاعلية وسجلات بث حي عبر WebSocket.

## المتطلبات
- Node.js 18+
- npm (أو pnpm/bun إذا فضلت)

## التثبيت والتشغيل المحلي
```sh
npm install
npm run dev
# افتح المتصفح على http://localhost:5173
```
إذا صادفت تعارضات peer dependencies استخدم:
```sh
npm install --legacy-peer-deps
```

## الإعدادات
- عنوان الـ API مضبوط على `https://slid.ethra2.com` داخل `src/lib/api.ts`. غيّره إلى خادمك (مثال: `http://127.0.0.1:8000`) عند الحاجة.
- WebSocket يتبع نفس الدومين ويستخدم `ws://` أو `wss://` تلقائياً حسب البروتوكول.
- البيانات التجريبية يمكن تفعيلها بتغيير `useMockAPI` في `src/lib/api.ts` إلى `true`.

## أوامر مفيدة
- `npm run dev` تشغيل محلي مع HMR.
- `npm run build` إنتاج نسخة جاهزة للنشر.
- `npm run preview` معاينة النسخة المبنية.
- `npm run lint` فحص الأكواد.

## ملامح الواجهات
- تسجيل دخول للمستخدمين الحكوميين.
- إدارة المستفيدين والجهات.
- إنشاء وإدارة طلبات التتبع (API/SMS) مع تحديث حالة الطلب.
- عرض تفاصيل الطلب مع خريطة تفاعلية ومسار السجلات، وتدفق حي عبر WebSocket لسجلات الموقع.

## بيانات تجريبية سريعة
- تسجيل دخول: جوال `0588888888` وكلمة مرور `password` (إذا كنت تستخدم الـ mock).
- إنشاء طلب: من صفحة "طلبات التتبع"، اختر مستفيداً وجهة والقناة ثم احفظ.
- عرض الطلب: افتح أي طلب لرؤية الخريطة والسجلات والتدفق الحي.
